Este es el primer proyecto del autor y aspira a crear un sistema de gestion de alquiler de vehículos.
Cuenta con un (backen)y un sitio web.
El lenguaje que se utilizo principalmente es PHP y Mysql
Se integro bootstrup //https://getbootstrap.com/docs/4.0/getting-started/introduction/
para la parte de interfaz de usuario.
Para su implementación se instalo xampp v 8.1.2 que es compatible con Mysql, Apache y MariaDB
El ambiente de trabajo fue realizado en Visual Studio Code donde se pudo utilizar algunas
extensiones y crear los directorios.
